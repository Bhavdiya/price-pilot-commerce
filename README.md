# 🛒 Price Pilot Commerce

**Price Pilot Commerce** is a modern, blazing-fast e-commerce frontend built with [Vite](https://vitejs.dev/), [React](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [TypeScript](https://www.typescriptlang.org/). Designed for scalability, responsiveness, and rapid development, it serves as a solid foundation for any commerce-based web application.

---

## 🚀 Tech Stack

| Tech            | Description                                 |
|-----------------|---------------------------------------------|
| Vite            | Lightning-fast frontend build tool          |
| React           | UI library for building interactive UIs     |
| TypeScript      | Strongly typed JavaScript                   |
| Tailwind CSS    | Utility-first CSS framework                 |
| PostCSS         | Used for Tailwind integration               |

## 🛠️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Bhavdiya/price-pilot-commerce.git
cd price-pilot-commerce/price-pilot-commerce-main
```
2. Install Dependencies
```bash
npm install
```
3. Run the Development Server
```bash
npm run dev
```
The app will be available at:
```bash
http://localhost:5173
```
